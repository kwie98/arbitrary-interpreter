Usage:
  arbipreter FILE [-p=PROGNAME] (-i=INPUT | -l=INPUTFILE) [-t]

Options:
  -p=PROGNAME   name of the program from the file to be run.
                Can be omitted if file only includes one program.
  -i=INPUT      use given input string for machine
  -l=INPUTFILE  read input from given file
  -t, --trace   output a trace
